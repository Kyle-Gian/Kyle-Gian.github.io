Showcase Piece
Demon King

I am currently working on my major project for my 2nd year at AIE, this team comprised of 2 Designers, 3 Artists and 2 Programmers. This game is built using the Unity Game Engine and is my first experience with networking. Demon King is an Online Multiplayer King of the Hill type game, where players need to hold the title of the Demon King for the longest time. The premise of the game is to gain power by consuming other players or AI, the mechanics used to create this experience had to be carefully planned due to the Networking aspect. This is a current work in progress and if you would like to find out more, please don't hesitate to contact me.

Node Based Behaviour Tree AI Unity Tool

This project was done during my 2nd year at AIE and is a Unity package that can be dragged into a project. The user can create a behaviour tree using a node based visual graph within the Unity engine. The user can easily use the script templates given "Node Check" & "New Behaviour" given to them to create any type of AI they require for their project. I will be looking to adding to this project in the future. A feature I am looking to add is a visual debugging tool, which shows which nodes and edges are active in runtime. If you would like to view this project on Github please click the icon or click the Unity icon to download the Unity package and add to your own project.
